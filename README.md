# Pakistani Women's Beauty Salon

A premium, dynamic web application for a Pakistani ladies' beauty parlor, featuring bridal makeup, mehndi, hair styling, skincare, and nail services.

## Features

- **Dynamic Branding:** Customize the salon name, tagline, and theme color via URL parameters (e.g., `?name=Elite+Beauty`).
- **Service Showcase:** Detailed categories and pricing for various beauty treatments.
- **Bridal Packages:** Specialized wedding service collections.
- **Gallery:** Visual showcase of salon expertise (Bridal, Mehndi, etc.).
- **Responsive Design:** Optimized for both mobile and desktop experiences.
- **Premium Aesthetics:** Elegant Urdu-inspired design with a sophisticated color palette.

## Getting Started

### Prerequisites

- Node.js & npm

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```

## Customization

You can dynamically change the site branding for different clients using URL parameters:

- `?name=Your+Salon+Name`
- `?tagline=Your+Custom+Tagline`
- `?color=HEX_CODE` (e.g., `color=E05270`)

Example: `https://your-domain.com/?name=Elite+Beauty&color=1A7A5E`
